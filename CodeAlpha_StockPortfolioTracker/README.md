# Stock Portfolio Tracker

## Description

A Python-based Stock Portfolio Tracker developed using Tkinter. This application allows users to add stock holdings, calculate investment values using predefined stock prices, and manage a simple investment portfolio through an interactive graphical user interface.

## Features

* Add stocks and quantities
* Hardcoded stock price dictionary
* Automatic investment value calculation
* Portfolio table display
* Total portfolio value calculation
* Remove selected holdings
* Clear portfolio
* Export reports as TXT files
* Export reports as CSV files
* Input validation and error handling

## Technologies Used

* Python 3
* Tkinter
* CSV Module
* File Handling

## How to Run

```bash
python stock_tracker.py
```
