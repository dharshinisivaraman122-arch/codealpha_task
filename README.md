# CodeAlpha Internship Project Submission

## Overview

This repository contains two Python projects developed as part of the CodeAlpha Internship Program. These projects demonstrate Python programming fundamentals, GUI development using Tkinter, file handling, dictionaries, conditional statements, functions, and object-oriented programming.

---

## Task 2: Stock Portfolio Tracker

### Description

A GUI-based Stock Portfolio Tracker built using Python and Tkinter. The application allows users to add stocks, calculate investment values, manage a portfolio, and save reports.

### Features

* Add stock symbols and quantities
* Calculate portfolio value
* Display holdings in a portfolio table
* Remove selected stocks
* Clear portfolio
* Save reports as TXT files
* Export data as CSV files
* Input validation and error handling

### Technologies Used

* Python 3
* Tkinter
* CSV Module
* Datetime Module

---

## Task 4: Rule-Based Chatbot

### Description

A GUI-based Rule-Based Chatbot developed using Python and Tkinter. The chatbot responds to user queries using predefined rules and provides an interactive conversational experience.

### Features

* Interactive chat interface
* Greeting responses
* Date and time information
* Python, AI, and Machine Learning explanations
* Random jokes and motivational quotes
* Help command
* Save chat history
* Clear chat functionality
* User-friendly GUI design

### Technologies Used

* Python 3
* Tkinter
* Datetime Module
* Random Module
