# Basic Chatbot

## Description

Basic Chatbot is a GUI-based chatbot developed using Python and Tkinter. The application allows users to interact with the chatbot through a simple graphical interface. The chatbot can respond to greetings, provide information about Python, Artificial Intelligence, and Machine Learning, tell jokes, share motivational quotes, display the current date and time, and save chat history.

This project was developed as part of the CodeAlpha Internship Program and demonstrates Python programming fundamentals, GUI development, event handling, and file handling.

## Features

* Interactive graphical user interface (GUI)
* Greeting and conversation support
* Information about Python
* Information about Artificial Intelligence (AI)
* Information about Machine Learning (ML)
* Random jokes and motivational quotes
* Current date and time responses
* Help menu with supported commands
* Save chat history as a TXT file
* Clear chat functionality
* User-friendly Tkinter interface

## Technologies Used

* Python 3
* Tkinter
* Datetime Module
* Random Module
* File Handling

## Concepts Demonstrated

* Conditional Statements (if-elif-else)
* Functions
* Lists and Strings
* Event-Driven Programming
* Input and Output
* File Handling
* GUI Development with Tkinter

## How to Run

1. Download or clone the repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python chatbot.py
```
